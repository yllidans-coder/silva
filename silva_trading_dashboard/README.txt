PACOTE PRONTO — SILVA TRADING DASHBOARD

1) Descompacte a pasta.
2) Envie tudo para um repositório no GitHub.
3) No Vercel, clique em "Add New Project".
4) Escolha o repositório.
5) Clique em Deploy.

Comandos locais, se precisar:
- npm install
- npm run dev
- npm run build

Formato esperado do arquivo:
Datetime | WIN | DOL | EWZ | ITUB4 | VALE3 | ITUB_US | VALE_US | PBR_A

Você pode subir CSV ou XLSX direto no painel.
